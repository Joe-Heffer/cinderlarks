Cinderlarks

Website concept sketches

Prototype designs for The Cinderlarks' website, each living in its own self-contained folder under sketches/.

  • cinderlarks-quiet-pages

    Literary "quiet pages" concept, now a single-page Tailwind hero landing (Playfair Display / Lora / Caveat, cream / dusty-pink / sage / gold palette) with About, Songs, Shows and Contact as scrolling chapters, plus five full song "chapter" pages to turn to for the deeper read.

  • drowned

    "The Kid Who Drowned" song page: a darker, submerged variant of the site's palette for the album's heaviest song.

  • rings-and-cups

    An alternate "Cups and Rings" concept: Neolithic cup-and-ring stone carvings, weathered grey / moss / ochre palette with a single blood-red accent, hand-carved typography and generous negative space.

  • steel-and-static

    A general band landing/hero page in an "industrial / analogue grit" aesthetic: Sheffield steel-city textures, rust and brass tones, and DIY tape/screen-print motifs as a bold backdrop for a still-gentle folk band.

  • still-life

    A single-page Tailwind hero prototype: cream / dusty-pink / sage / plum "quiet pages" palette, Fraunces / Public Sans / Caveat type, hero treated as a still life of hand-drawn song motifs (ring, jug, envelope, fossil, clock) instead of a band photo.

  • template

    Starting point for a new sketch — not a real design.

No build step, no framework — just static HTML, CSS and JS, deployed via GitHub Pages.
